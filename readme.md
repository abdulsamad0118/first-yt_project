# YouTube Video Manager

A simple command-line application built with Python and SQLite to manage a list of videos.

## About

I created this project to practice working with SQLite databases in Python. The application allows users to add, view, update, and delete video records through a menu-driven interface.

## Features

* View all videos
* Add a new video
* Update existing video details
* Delete a video
* Store data using SQLite

## Technologies Used

* Python
* SQLite3

## Running the Project

Clone the repository and run the Python file:

```bash
python youtube_manager.py
```

## Database

The application automatically creates a SQLite database named `youtube_video.db` and stores all video information in a table called `videos`.
